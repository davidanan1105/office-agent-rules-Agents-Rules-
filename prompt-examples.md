# 常用提示词示例

## 1. 让 AI 先出计划，不直接动手

```text
请按 AGENTS.md 的规则执行。先不要修改文件。

请先说明：
1. 你理解的任务目标；
2. 准备处理哪些文件、表格或段落；
3. 计划采用的最小可行方案；
4. 有哪些假设和风险；
5. 完成后我应该如何验收。
```

## 2. 处理表格

```text
请处理这份表格，但不要覆盖原文件。

要求：
1. 先识别 sheet、字段、公式和关键数据；
2. 说明需要修改哪些位置；
3. 保留原始数据；
4. 输出一个新文件；
5. 完成后列出修改内容、校验方式和剩余风险。
```

## 3. 整理制度或通知

```text
请把这份材料整理成更清晰的版本。

要求：
1. 不改变原始业务口径；
2. 不修改数字、时间、对象范围和责任部门；
3. 可以优化结构和标题层级；
4. 如发现歧义，只标注风险，不自行补充事实；
5. 输出修改摘要和待确认问题。
```

## 4. 生成交接材料

```text
请基于当前内容生成一份交接说明。

要求：
1. 使用“任务目标、背景约束、已处理对象、关键处理逻辑、已完成内容、验证情况、未验证范围、剩余风险、下一步动作”的结构；
2. 不增加未经确认的事实；
3. 对不确定内容明确标注；
4. 输出简洁、可给下一个人继续使用。
```

## 5. 轻量页面或 Markdown 展示

```text
请把以下内容整理成一个可阅读的 Markdown 页面。

要求：
1. 保留原始事实、数字和责任边界；
2. 优化标题层级和阅读顺序；
3. 不添加原文没有的承诺或解释；
4. 完成后说明主要结构调整。
```
